ThemeKings Military Clan v2.0 | Thank You For Downloading

 To change the banner picture you can open the .png file located in the THEME folder titled "banner-blank.png". This is the original layered file I used to design your current banner image titled "banner-blank.jpg". I used Adobe Fireworks CS5.5 to create this image. If you choose to change or update this banner please feel free to use this picture file to make any changes you like. Once you do so, overwrite the current .jpg file with your own. If you give it another name, be sure to change the banner name and file extension in the CSS file. The ID for the banner is simply "#banner".

 The themes primary CSS file is titled "military-styles.css". Here you will find all the classes and id's needed to style this webpage with exception to the horizontal navigation menu above. That CSS file is titled "navigation.css".

 This theme includes a song by C Murder titled 2nd Chance. Place all new .mp3 files into the "songs" folder. To update or add more songs open the "pi_player_playlist.xml" file located in the main root directory. Starting at line 51 in this XML file you'll see a section of code contained within the "song" DIV. You can copy those 6 lines of code and paste them below the other and update the song information to easily add your next song in your play list. At the top of the "pi_player_playlist.xml" I've commented on certain elements of the MP3 player and how changing those elements will make changes to how the player acts. I personally wouldn't mess with those for this template, but if you was curious to know there you be.

 If you choose to use the TWITTER feed, all you need to do use update the username, to your twitter username. By default I set this one to g4tv. Just change that to yours and that's it. Instant feed! That can be located near the bottom of the twitter feed script for that vertical block.

 Any other questions or concerns please register on the ThemeKings.Net website and ask your questions on the download page for this template. Thank you and enjoy!
